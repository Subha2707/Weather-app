# 🌦️ Weather App

A modern, responsive weather application built using **HTML, CSS, and JavaScript** that provides real-time weather data, 7-day forecasts, and interactive visualizations.

---

## 🚀 Features

* 📍 **Auto Location Detection** (Geolocation API)
* 🔍 **Search Any City**
* 🌡️ **Real-Time Weather Data**
* 🌧️ **7-Day Forecast**
* 📊 **Animated Hourly Temperature Graph (Canvas)**
* 🎨 **Dynamic Background Based on Weather**
* 🌙 **Dark Mode Toggle**
* ❄️ **Rain & Snow Animations**
* 💎 **Glassmorphism UI Design**
* 📱 **Fully Responsive (Mobile + Desktop)**

---

## 🛠️ Tech Stack

* **HTML5**
* **CSS3** (Glassmorphism + Responsive Design)
* **JavaScript (ES6)**
* **OpenWeatherMap API**

## ⚙️ Setup Instructions

### 1. Clone the repository

```
git clone https://github.com/your-username/weather-app.git
cd weather-app
```

---

### 2. Get API Key

* Go to: https://openweathermap.org/api
* Sign up and generate a free API key

---

### 3. Add API Key

Open `script.js` and replace:

```
const apiKey = "YOUR_API_KEY";
```

with your actual API key.

---

### 4. Run the Project

#### Option 1: Live Server (VS Code)

* Install Live Server extension
* Right click `index.html` → **Open with Live Server**

#### Option 2: Python Server

```
python -m http.server
```

Then open:

```
http://localhost:8000
```

---

## 🔒 Security Note

* Do **NOT expose your API key publicly**
* Replace API key with `"YOUR_API_KEY"` before pushing to GitHub
* For production, use a backend to secure API keys
